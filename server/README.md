## Collaborative Markdown

Google docs clone that enables users to have a collaborative editor for creating Markdowns for documentations

___

This ReadMe.md itself is made using the websiteÂ 

Using **React.js + Typscript** for the frontend for a simple yet understandable UI and **Node.js** for the backend for the websocket that enables collaborative document editing

___

## Features

- Real-time collaborative editing using WebSockets

- Live Markdown preview

- Simple and minimalistic UI

- Lightweight and easy to extend

---

## Tech Stack

- **Frontend:** React, TypeScript, Vite

- **Backend:** Node.js, WebSocket (ws)

- **Markdown Rendering:** marked
